# Society_Of_Minds_Autogen_CrewAI_StreamlitUI
Streamlit UI based ChatBot for showcasing Society Of Minds Concept using Autogen and CrewAI Frameworks in python v3.11.0
- This repo consists of scripts and outputs of chatbot in PDF format
- Modify the paths in the scripts as required
- Script uses groq LLM(llama)
- Install packages as "python.exe -m pip install requirements.txt"
- Run the python scripts as "python.exe -m streamlit run pythonfile.py"
